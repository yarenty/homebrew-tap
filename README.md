# Yarenty Tap

## How do I install these formulae?

`brew install yarenty/tap/<formula>`

Or `brew tap yarenty/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "yarenty/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
